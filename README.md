# react-native-stripe-apple-pay

React Native wrapper around Stripe Apple Pay

## Installation

```sh
npm install react-native-stripe-apple-pay
```

## Usage

```js
import { multiply } from 'react-native-stripe-apple-pay';

// ...

const result = await multiply(3, 7);
```
